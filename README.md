# 🤖 AI Chatbot using Groq & LLaMA-3

## 📌 Overview

This project is an **AI-powered chatbot** that enables users to interact through natural language conversations. It is designed to answer questions, assist users, and provide intelligent responses in real time.

The chatbot leverages the power of **Groq API** and the **LLaMA-3 model**, ensuring fast and efficient response generation with strong language understanding.

---

## 🚀 Features

* 💬 AI-powered conversational chatbot
* ⚡ Ultra-fast responses using Groq inference
* 🌐 Interactive web-based chat interface
* 🔗 Seamless frontend-backend integration
* 🧠 Natural language understanding with LLaMA-3

---

## 🛠️ Tech Stack

* **Backend:** Python
* **Frontend:** HTML, CSS
* **AI Model:** LLaMA-3
* **API:** Groq API

---

## ⚙️ How It Works

1. User sends a message through the chat interface
2. The request is processed by the Python backend
3. Backend sends the query to Groq API
4. LLaMA-3 generates a response
5. Response is returned and displayed in the chat UI

## 🔧 Setup Instructions

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Tanishqbobde/AI-chatbot.git
cd AI-chatbot
```

---

### 2️⃣ Create Virtual Environment (Important ⚠️)

```bash
python -m venv venv
```

Activate it:

* **Windows**

```bash
venv\Scripts\activate
```

* **Mac/Linux**

```bash
source venv/bin/activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Add Groq API Key

Create a `.env` file and add:

```
GROQ_API_KEY=your_api_key_here
```

---

### 5️⃣ Run the Application

```bash
python app.py
```
